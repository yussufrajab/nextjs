CSMS BULK EMPLOYEE UPLOAD - QUICK START GUIDE
==============================================

For Human Resource Officers (HROs)

STEP 1: DOWNLOAD TEMPLATE
--------------------------
Download: CSMS_Employee_Bulk_Upload_Template.csv

STEP 2: FILL IN EMPLOYEE DATA
------------------------------
Open the CSV file in Excel or Google Sheets

Required Fields (Must fill):
- Name
- Gender (Male or Female)
- ZanID (unique)
- Date of Birth (format: YYYY-MM-DD, example: 1990-01-15)
- ZSSF Number (unique)
- Payroll Number (unique)
- Cadre (job title)
- Ministry
- Department
- Employment Date (format: YYYY-MM-DD)

Optional Fields:
- Place of Birth
- Region
- Country of Birth
- Phone Number (format: 0XXXXXXXXX, example: 0773101012)
- Contact Address
- Salary Scale
- Appointment Type
- Contract Type
- Recent Title Date
- Current Reporting Office
- Current Workplace
- Confirmation Date
- Retirement Date
- Status (default: On Probation)

IMPORTANT FORMATS:
- Dates: YYYY-MM-DD (e.g., 1990-01-15)
- Phone: 0XXXXXXXXX (e.g., 0773101012)
- Gender: Male or Female (case-sensitive)

DELETE THE SAMPLE ROWS before adding your data!

STEP 3: SAVE AS CSV
--------------------
File → Save As → CSV (Comma delimited)

STEP 4: UPLOAD TO CSMS
-----------------------
1. Login to https://csms.zanajira.go.tz
2. Go to Dashboard → Add Employee
3. Click "Bulk Upload" tab
4. Choose your CSV file
5. Click "Validate File"
6. Review validation results
7. Fix any errors if needed
8. Click "Upload X Employee(s)"
9. Done!

COMMON ERRORS:
--------------
1. "Invalid date format" → Use YYYY-MM-DD
2. "Phone number must be 10 digits" → Use 0XXXXXXXXX
3. "Duplicate ZanID" → ZanID already exists, use different one
4. "Name is required" → Fill in the Name field
